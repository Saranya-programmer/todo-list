# 📝 To-Do List

A simple and interactive To-Do List web application built using HTML, CSS, and JavaScript.

## 🚀 Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Filter tasks (All / Completed / Pending)
- Clear all tasks
- Data persistence using localStorage

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript

## 🧠 Key Concepts Used

- DOM Manipulation
- Event Handling
- Arrays & Objects
- localStorage
- Dynamic UI Rendering

## 💡 How to Use

1. Enter a task in the input field
2. Click "Add" or press Enter
3. Manage tasks using edit, delete, or complete options

## 📌 Future Improvements

- Add due dates
- Add priority levels
- Drag & drop tasks
